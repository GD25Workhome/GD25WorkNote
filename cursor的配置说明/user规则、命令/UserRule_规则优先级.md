# User Rule：规则优先级和说明

## 规则内容

```
优先级顺序：
1. Conda 环境正确性 > 其他所有规则
2. 用户明确要求 > 默认规则（如文档生成规则）
3. 错误修复 > 功能开发
4. 版本兼容性 > 使用最新版本

特殊说明：
- 这些规则应该始终被遵守，即使在长对话中
- 如果规则之间存在冲突，优先处理 conda 环境相关的规则
- 如果发现规则不合理或需要更新，应该告知用户并请求确认
```

## 使用说明

此规则定义了所有其他规则的优先级顺序，是 Cursor AI 在决策时的最高指导原则。

**适用场景**：
- 当多个规则发生冲突时
- 需要决定执行顺序时
- 需要判断优先级时
